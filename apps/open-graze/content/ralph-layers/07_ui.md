# 07 UI

Presentation Data를 최종 사용자 인터페이스로 렌더링한다.  
UI는 Need 수행을 지원하고, 다음 Need를 다시 입력/수집하며 루프를 반복한다.
이 단계의 트리거는 직전 단계(06) 문서의 미완 체크리스트를 따른다.

## Checklist (작성자: 06 Presentation Data 단계)

- [x] Presentation Data를 카드/상태 배지/최신 이벤트/추천 텍스트로 일관되게 렌더링한다.
- [x] 다음 Need/Action으로 이어지는 입력(앱 니즈 입력)과 추천 동선(recommendation)을 제공한다.
- [x] Presentation Data의 updatedAt/completeness를 UI에서 함께 노출해 데이터 신뢰 신호를 제공한다.
